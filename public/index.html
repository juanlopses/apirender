Documentación de la API Media Scraper

TikTok Video Info (GET /api/ttdl)

Ejemplo: /api/ttdl?url=https://vt.tiktok.com/xxxx

{
  "title": "Video title",
  "author": "Author name",
  "video": "direct_video_url",
  // ... otros campos
}

YouTube MP3 (GET /api/ytmp3)

Ejemplo: /api/ytmp3?url=https://youtube.com/watch?v=XXXX

YouTube MP4 (GET /api/ytmp4)

Ejemplo: /api/ytmp4?url=https://youtube.com/watch?v=XXXX

Instagram Download (GET /api/igdl)

Ejemplo: /api/igdl?url=https://instagram.com/p/XXXX

Facebook Download (GET /api/fbdl)

Ejemplo: /api/fbdl?url=https://facebook.com/XXXX

YouTube Search (GET /api/ytsearch)

Ejemplo: /api/ytsearch?query=nombre+canción

Uso básico con JavaScript:

fetch('/api/ttdl?url=TU_URL')
  .then(response => response.json())
  .then(data => console.log(data));

Limitaciones:

  • Límite de 100 solicitudes cada 15 minutos
  • Solo métodos GET
  • Debe incluir el parámetro URL en la consulta
